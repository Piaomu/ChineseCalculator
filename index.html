ChineseCalculator

好好学习，更好编程

This simple calculator translates your calculations (between 1 - 100) into Chinese script.
Numbers in Chinese are logically sequenced like a hybrid between Arabic numerals and Roman numerals. Unlike the Arabic System, the written form indicates the number of units in each base together with that base.
For example, 23 is actually written as "two tens and three" (二十三).


Parameters of the project:

  • Produce an application that has a textbox on it that takes a number. Create four buttons, one for each addition, subtraction, multiplication, and division. Then, create a button that performs the desired operation upon clicking.


Additional Functionality:

  • Create a feature that translates the inputs, operations, and results into understandable Chinese for non-Chinese speakers.
  • Write the translation logic such that it mirrors the logic of the Chinese written system.
